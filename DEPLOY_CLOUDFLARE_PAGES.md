# Deploy Zeruva Aliens Frontend to Cloudflare Pages

This repo is a Next.js app. Cloudflare Pages deployment uses `@cloudflare/next-on-pages`.

## Prereqs
- Cloudflare account
- `wrangler` installed (optional but recommended)

## Build command
Use:

```bash
npm run build:cf
```

## Output directory
Cloudflare Pages should deploy the `.vercel/output/static` output generated by next-on-pages.

## Pages settings (Dashboard)
- **Framework preset:** Next.js
- **Build command:** `npm run build:cf`
- **Build output directory:** `.vercel/output/static`

## Environment variables
Set in Cloudflare Pages → Settings → Environment variables:
- `NEXT_PUBLIC_API_BASE_URL` (optional)
  - Leave unset if you rely on `next.config.mjs` rewrites to Railway.

## Notes
- API calls should go to same-origin `/api/*` and be rewritten to Railway (see `next.config.mjs`).
- If you later change backend URL, update the rewrite destination.
